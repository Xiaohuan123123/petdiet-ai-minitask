# 第4章：商业判断（核心结论）

---

## 4.0 先定义清楚：我们在服务谁？

在讨论怎么赚钱之前，先把人找对。

市面上绝大多数宠物 AI 产品都在做"看病"——在线问诊、症状分析、用药提醒。这不是我们要做的事。

我们要服务的是这样一群人：

> 他们不是宠物生病了才来找答案。
> 他们**主动想给宠物做对吃的、做好吃的、买对吃的**。
> 他们可能是想做鲜粮但不知道配方，可能是想做自制但怕营养不均衡，可能是想买粮但不知道哪个品牌靠谱。
> 他们的共同点是：**愿意为宠物的饮食花心思，但缺乏专业工具帮他们做对决策。**

用一个词概括：**饮食 DIY 型宠主**。

这群人的规模比"病后宠主"大得多。白皮书数据显示：鲜粮购买占比在持续上升（犬鲜粮 25.7%、+1.7pp）；冻干粮、烘焙粮、湿粮等"非传统膨化粮"品类全面上升；高消费养宠人群占比 26.7% 且持续增长。这些信号都在说同一件事：**越来越多的宠主不再满足于"倒一碗狗粮就完事"，他们想做得更好。**

这就是宠食记的核心用户。

## 4.1 谁会为这个产品付钱？

围绕"饮食规划 + DIY 制作 + 购买推荐"这条主线，拆出四类付费用户：

| 用户层 | 典型画像 | 核心场景 | 付费意愿 |
|------|------|------|:---:|
| **鲜粮 DIY 型** | 想做鲜粮/自制但不知道怎么配。买食材的时候不确定哪种安全、哪种比例对 | "给我一份针对我家狗的鲜粮食谱，告诉我去哪买这些食材" | ⭐⭐⭐⭐⭐ |
| **品质升级型** | 已经在喂高端粮，但想知道有没有更好的选择。关注成分表、关注品牌口碑、愿意为品质付溢价 | "帮我看看这个粮的成分好不好，有没有更适合我家宠物的" | ⭐⭐⭐⭐ |
| **日常规划型** | 想给宠物科学喂养但不确定每天喂多少、喂几次、什么时候换粮。不是出了事才问，而是日常想做好 | "帮我规划一下本周的饮食安排" | ⭐⭐⭐ |
| **焦虑解惑型** | 在网上搜到一堆相互矛盾的信息，需要一个可信的来源帮他们判断 | "XX食材能不能给我的猫吃？""换粮软便了怎么办？" | ⭐⭐⭐⭐ |

**早期核心付费用户是鲜粮 DIY 型。**

为什么？因为他们的决策链路最短：想做鲜粮 → 需要食谱 → 需要食材清单 → 需要知道去哪买。这个链条上的每一步，都是可以被付费产品化的。而且这个场景目前市场上几乎是空白的——小红书上有零散的食谱分享，但不针对你的宠物；宠物营养师可以定制但贵且少；通用 AI 可以生成食谱但不帮你买食材。

## 4.2 用户为什么付费？

用户不会为"AI 很酷"付钱。他们付钱是因为这个产品解决了他们具体到不能再具体的需求。

用 Jobs-to-Be-Done 来描述：

| JTBD（用户雇佣产品要完成的任务） | 不用宠食记的替代方案 | 宠食记的方案 | 为什么愿意付钱 |
|------|------|------|------|
| **"给我的宠物做一顿营养均衡的鲜粮/自制"** | 小红书搜食谱（不针对我的宠物、不知道安不安全）、问宠物群（回答质量参差不齐）、自己研究（花时间且不一定对） | **输入宠物档案 → AI 生成专属食谱 → 食材清单 + 分量 + 步骤 + 营养补充提示** | 这是从"想做"到"做对"的关键一步。替代方案太不可靠 |
| **"我不知道该给我的宠物买什么粮"** | 看电商评价（真假难辨）、问朋友（别人家的狗和我的不一样）、凭感觉买 | **AI 基于宠物档案 + 成分分析，推荐适合的粮 + 解释为什么** | 解决选择困难，避免买错浪费钱 |
| **"我想知道XX食材能不能喂、怎么喂"** | 百度搜索结果互相矛盾、小红书上有人说能吃有人说不能吃 | **AI 给出明确的安全判断 + 喂食建议 + 注意事项** | 为确定性付费。喂错可能伤害宠物，信息不对称的焦虑很强 |
| **"帮我规划这周/这个月的宠物饮食"** | 没有替代方案，大部分人是"想到什么喂什么" | **基于宠物档案 + 目标（减肥/增重/维持），生成周度/月度饮食计划** | 这是新增的需求。用户可能不知道有这个需求，但一旦体验过就回不去了 |
| **"根据菜谱，一键帮我买齐食材"** | 自己对着食材清单一个个搜、比价、下单 | **AI 自动匹配食材 → 加入购物车 → 一次性下单** | 省时间。做了食谱但懒得一个个买食材，这是真实存在的摩擦 |

**核心洞察：用户是在为"从想做→做成"这件事付费。** "想做鲜粮"和"做出一顿营养均衡的鲜粮"之间，有一个很大的 gap。宠食记填的就是这个 gap。

## 4.3 如何定价？

### 4.3.1 定价原则

1. **按"任务完成度"定价，不按功能定价**：用户买的是"一份我可以直接照着做的食谱"，不是"AI 对话次数"
2. **低门槛让用户第一次付费没有心理负担**：¥9.9 和 ¥199 是完全不同的决策
3. **锚定参照系让付费看起来划算**：对比宠物营养师定制（¥100-300/次），AI 方案 ¥9.9-19.9 是白菜价
4. **先让用户感受到"AI懂我的宠物"，再谈钱**：免费版充分展示个性化能力，付费是对"更深度的服务"买单

## 4.4 如何获客？

第一阶段不投钱。用内容和产品本身撬动自然流量。

### 渠道一：小红书 —— 用"AI食谱"种草（主力）

这是最适合宠食记的获客渠道。小红书宠主月活 8200 万，而且用户心智就是"种草 + 学习"。

**内容策略**：不做硬广，做"真实 AI 食谱展示"。

每条内容的格式：
- 标题："给 3 岁金毛定制的南瓜鸡肉粥食谱（附食材清单）"
- 配图：宠食记 APP 截图（宠物档案卡 + AI 生成的食谱表格 + 食材清单）
- 正文：简述这只宠物的情况 → AI 分析了什么 → 给出了什么方案 → 评价是否靠谱
- 结尾钩子："想给你的宠物也生成专属食谱？评论区告诉我它的品种/体重/年龄，我帮你跑一份"

**选题库**：
- "12kg 柯基一顿该吃多少？AI 帮算出来了"
- "鸡肉过敏的金毛能吃什么？AI 给了 5 个替代方案"
- "想做鲜粮不知道配比？AI 一键生成完整食谱"
- "把狗粮成分表丢给 AI 分析，结果吓我一跳"

**节奏**：第一周每天 3-4 条，第二周根据数据反馈调整频率和选题方向。

**目标**：两周内积累 30-50 条真实评论追问（不是"好棒"这种，而是"帮我家的也算一下"这种）。

### 渠道二：宠物社群 —— 用"免费帮算"裂变

加入 10-15 个活跃宠物群（品种群 + 地区群 + 鲜粮/自制兴趣群）。

**不是去打广告。** 而是看到有人在群里问"我家狗吃什么好""想尝试鲜粮不知道怎么配""XX 狗粮好不好"时：
1. 用宠食记帮 TA 生成一份回答
2. 截图发群里："帮你用 AI 跑了一下，参考看看"
3. 不推 APP，不推链接。等有人主动问"这是什么工具？"
4. 这次再自然地给链接

**关键心态**：先提供价值，再获得关注。如果发 10 次有价值的回复，有 3 个人追过来问"这是哪个 APP"，那就是成功的。

**目标**：两周内至少在 10 个群里建立"那个帮大家算食谱的人"的身份认知。

### 渠道三：宠物店/鲜粮食材店合作 —— 线下触点

谈 3-5 家本地宠物店，特别是那些有卖鲜粮食材（冷冻肉、蔬菜包、补剂）的店。

**合作逻辑**：店铺帮我们放一个立牌，二维码扫进去是宠食记。反过来，宠食记在食谱的"食材购买"环节推荐这家店的食材配送。互利共赢，不花钱。

**话术**："你家卖鲜粮食材，但很多客户买了不知道怎么配。我们用 AI 帮你的客户出食谱，顺便推荐你家的食材。你的客户获得了专业服务，你多卖了食材。"

**目标**：找到 2-3 家愿意合作的店。

### 两周获客目标

| 指标 | 目标值 | 重要性 |
|------|:---:|:---:|
| 小红书内容 | 30-40 条 | 参考 |
| 小红书真实评论追问 | 30-50 条 | **核心** |
| 社群主动询问"这是什么" | 15-20 次 | **核心** |
| 合作店铺 | 2-3 家 | 参考 |
| 产品注册用户 | 300-800 人 | 参考 |
| 活跃对话用户（>5 轮） | 50-100 人 | **核心** |
| 对话完成率 | > 60% | **核心** |

**最关键的不是注册量。是有多少人用了之后觉得"这个东西对我是有用的"。**

## 5.1 PEST：为什么这个时机是对的？

| 维度 | 关键发现 | 意味着什么 |
|------|------|------|
| **P 政治** | 宠物行业政策友好（高铁托运 8 线 25 站、宠物航站楼等）；AI 监管边界清晰 | ✅ |
| **E 经济** | 宠物消费逆周期 3126 亿/+4.1%；冻干粮/烘焙粮/鲜粮全面上升；高消费人群 26.7%/+3.5pp；AI API 成本极低 | ✅ 市场在涨、品质升级趋势明确、技术成本在降 |
| **S 社会** | 宠物家人化（"家人"角色回升）；科学养宠觉醒（61% 关注知识干货）；90后+00后占 70%，独居经济 | ✅ 用户已经在主动学、主动升级 |
| **T 技术** | DeepSeek-V4-Pro Agent 开源第1、1M 上下文中文天花板；多模态快速补上；RAG 技术成熟 | ✅ 能力够了，成本降了，巨头还没做这个垂域 |

**综合：四维利好。当前是 AI 宠物饮食规划产品的最佳时间窗口。**

## 5.2 波特五力：为什么这个垂域值得做？

整个宠物经济是红海——但在"AI 宠物饮食规划+DIY制作+购买推荐"这个垂域——**没有直接竞品**。

| 五力 | 强度 | 判断 |
|------|:---:|------|
| **现有竞争者** | 🟢 低 | 同是 AI 宠物产品的绮算法、汪喵灵灵都偏"医疗诊断 B 端"，不做"饮食 DIY C 端"。最接近的替代品是小红书上的食谱分享，但零散、不个性化 |
| **新进入者** | 🟡 中 | 技术门槛低（调 API 谁都会）。但用户信任、数据积累、Prompt 经验需要时间 |
| **替代品** | 🔴 中高 | ⚠️ **通用 AI 是最需要注意的替代品**。用户可以直接问 DeepSeek/Kimi。防守点：宠物档案持久化 + 历史上下文 + 垂域 Prompt + 结构化食谱 UI + 食材购买闭环——这五件通用 AI App 做不到 |
| **供应商** | 🟢 低 | DeepSeek 在降价，GLM/Qwen 随时可替代 |
| **买家** | 🟢 低 | C 端用户不讨价还价。真实压力是"用户可以不付钱" |

## 5.4 麦肯锡矩阵：为什么这条路径？

```
              市场吸引力
              高             低
          ┌──────────────┬──────────────┐
    强    │ 🎯 鲜粮DIY食谱 │ 📊 成分解码      │
    竞    │   周度饮食规划  │   安全查询       │
    争    ├──────────────┼──────────────┤
    地    │              │              │
    位    │ ⚡ 食材购买闭环 │ 🚫 宠物社交      │
    弱    │   （长期建设）  │   医疗问诊       │
          └──────────────┴──────────────┘
```

**破局路径**：

1. **阶段 1（2 周）**：以"免费 AI 食谱生成"建立初始用户群。证明有人愿意让 AI 帮他们规划宠物饮食
2. **阶段 2（4-8 周）**：食谱场景跑通后用单次付费（¥9.9-19.9）验证付费。证明有人愿意为深度食谱/规划付钱
3. **阶段 3（3-6 月）**：推进订阅（¥29-49/月）。证明"饮食规划"不是一次性需求，而是持续需要
4. **阶段 4（6-12 月）**：打通食材购买闭环。从"AI 告诉你买什么"到"AI 帮你一键买齐"

**一句话**：**用免费食谱建立信任 → 用深度食谱验证付费 → 用订阅锁定长期价值 → 用购买闭环完成商业变现**。

## 5.5 贝叶斯财务模型：滚动预测的方法论

（详见第 4.5 节收入预测部分）

核心逻辑：不是一次性"预测 6 个月收入 = ¥3,300"，而是每拿到一个新数据就滚动更新。两周验证拿到留存率和转化率 → 第一次更新 → 3 个月后如果出现内容爆款 → 第二次更新 → 如果订阅验证通过 → 第三次更新。财务模型是活的，不是死的 Excel 表。
